# Step-500 Samples

This directory contains generated samples and logs from the step-500 smoke run.

Included:

- `generated_samples/*.wav`: generated Gemma-to-Mimi outputs.
- `metadata.json`: generation metadata.
- `sample_summary.json`: ASR overlap summary.
- `asr.json`: raw ASR evaluation output.
- `head_config.json`: architecture config for the trained head.

Not included:

- `mimi-head-step-500.pt`
- full Gemma caches
